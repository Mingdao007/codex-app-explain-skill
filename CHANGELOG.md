# Changelog

## 0.1.0 - 2026-03-12

- First public release of `codex-app-explain`
- Extracted a generic `Codex App` layered-teaching workflow from a private
  local skill
- Removed personal paths, private memory bindings, and private workflow names
- Added public references for teaching principles, source binding, and route
  adaptation

## 0.2.0 - 2026-03-14

- Added a public explanation-mode split:
  - `short-concept mode`
  - `prerequisite-chain mode`
  - `paper-sentence mode`
  - `full-layered mode`
- Added a public reference for explanation-mode selection
- Clarified the companion boundary between teaching routing and host-owned
  formatting or rendering rules

## 0.3.0 - 2026-03-14

- Expanded the public `paper-sentence mode` contract to include:
  - quoted-source Unicode-friendly cleanup as display-only behavior
  - paragraph-level and section-level synthesis as public reading support
- Added the public logic-role opening rule:
  - start with a minimal plain example close to the source sentence pattern
  - then give the direct role judgment
  - then return to the formal object
- Added public divider guidance for long explanations at clear structural
  transitions
- Aligned the root README and Chinese README to the same section skeleton used
  by the companion rules repository
