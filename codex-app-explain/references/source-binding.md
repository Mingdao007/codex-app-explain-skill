# Source Binding

Use this reference when the request depends on a specific equation, page,
passage, or bounded homework step.

## Bind these three items before explaining

- object: what is being explained
- source: where it comes from
- scope: which conditions or interpretation apply

## Binding order

1. Name the object
2. Name the source fragment
3. State the scope boundary

## Scope examples

- exact vs approximate relation
- special case vs general case
- local step vs full derivation
- one page vs one subsection

## Minimal prerequisite rule

- If the current blocker depends on one missing prerequisite, insert that
  prerequisite first.
- Keep it minimal and say why it is being introduced.

## Evidence rule

- Load the exact fragment needed for the explanation.
- Prefer local fragments over whole-document summaries.
